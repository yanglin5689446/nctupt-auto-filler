## nctupt auto filler

## Install
```
yarn
```

## Config
pt.config.json
```json
{ 
    "account": <account>, 
    "password": <password>,
    "works": {
        "PTSerialNo": <PTSerialnum>,
        "PTEId": <PTEId>,
        "workP": "107I113"
    },
    "month": "2018-12",
    "from": 3,
    "to": 28,
    "total": "110" 
}
```

## run
```
node index.js
```
